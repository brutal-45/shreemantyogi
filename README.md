## 🚀 **Getting Started**

### Prerequisites

```bash
node >= 18.0.0
bun or npm
```

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/shreemantyogi.git
   cd shreemantyogi
   ```

2. **Install dependencies**
   ```bash
   bun install
   # or
   npm install
   ```

3. **Run the development server**
   ```bash
   bun run dev
   # or
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

## 🎬 **Animations & Effects**

<div align="center">

| Animation | Type | Description |
|-----------|------|-------------|
| 🏳️ **Flag Wave** | CSS Keyframes | Realistic fabric wave effect on the Bhagwa Dhwaj |
| ✨ **Floating Particles** | CSS + JS | Golden dust particles floating upward |
| 🔆 **Golden Pulse** | CSS Keyframes | Glowing pulse effect on CTA buttons |
| 🌟 **Shimmer** | CSS Keyframes | Light reflection sweep across buttons |
| 📜 **Reveal on Scroll** | Intersection Observer | Elements fade in as they enter viewport |
| 💫 **Ring Pulse** | CSS Keyframes | Pulsing border around portrait |
| 🎯 **Spotlight Rotate** | CSS Keyframes | Rotating gradient glow effect |
| 🌙 **Theme Transition** | CSS Transition | Smooth color transitions on theme change |

</div>

<br/>

### CSS Animation Examples

<details>
<summary><b>🎬 View Animation Code</b></summary>

```css
/* Flag Wave Animation */
@keyframes flagWave {
  0%, 100% { transform: skewY(0deg) scaleX(1); }
  25% { transform: skewY(2deg) scaleX(0.98); }
  50% { transform: skewY(-1deg) scaleX(1.02); }
  75% { transform: skewY(1deg) scaleX(0.99); }
}

/* Golden Pulse */
@keyframes goldenPulse {
  0%, 100% { box-shadow: 0 0 30px rgba(212, 168, 75, 0.4); }
  50% { box-shadow: 0 0 50px rgba(212, 168, 75, 0.6); }
}

/* Shimmer Effect */
@keyframes shimmer {
  0%, 100% { transform: rotate(45deg) translateX(-100%); }
  50% { transform: rotate(45deg) translateX(100%); }
}

/* Floating Particles */
@keyframes floatUp {
  0% { opacity: 0; transform: translateY(0) scale(0); }
  10% { opacity: 0.8; transform: scale(1); }
  100% { opacity: 0; transform: translateY(-100vh) scale(0.5); }
}
```

</details>
